# Modelando-um-Dashboard-de-E-commerce-com-Power-BI-Utilizando-F-rmulas-DAX

  - Primeiramente foi criada uma chave ID_Vendas já que cada venda é unica
  
  - Então foram refeitas as tabelas: D_Detalhes, D_Produtos_Detalhes, e D_Produtos
  
  - A tabela D_Descontos não foi criada, pois os descontos fazem mais sentido na tabela de Vendas ou Detalhes
  
  - Criada a tabela Datas com as datas e a chave da venda
  
  - Com a função DAX foram criadas as colunas para ano mês e dia.
